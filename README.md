## Introduction

Croct is a headless CMS that helps you manage content, run AB tests, and personalize experiences without the hassle of complex integrations.

This SDK plugs Croct into [Shopify Hydrogen](https://hydrogen.shopify.dev) storefronts running on Oxygen. It works with both React Router 7 and Remix-era Hydrogen apps, personalizes content server-side in your loaders (no flicker), and automatically forwards Shopify analytics events to Croct.

## Installation

Run this command to install the SDK:

```sh
npm install @croct/plug-hydrogen
```

See our [quick start guide](https://docs.croct.com/reference/sdk/hydrogen/integration) for more details.
